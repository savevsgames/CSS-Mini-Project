FLEXBOX ROW

Use these properties to create a FLEXBOX ROW layout.

            
.row { 
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

FLEXBOX COLUMN

Use these properties to create a FLEXBOX COLUMN layout.

            
.column {
    display: flex;
    flex-direction: column;
}

CSS GRID

Build a 12 column grid with CSS.

            
.grid {
    display: grid;
    width: 100%;
    grid-template-columns: repeat(12, 1fr);
    gap: 1rem;
}

LINEAR GRADIENTS

Create a linear background gradient from top to bottom.

            
.linear-gradient-background {
    background-image: linear-gradient { 
        rgba(232, 162, 236, 0.3) 0%,
        rgba(162, 236, 232, 0.6) 100%
    };
}

BOX TRANSITION GLOW

Use transition and box shadows to glow on hover.

            
.code-card .card-header {
    border-radius: 10px;
    transition: all 0.5s ease-in-out;
}

.code-card:hover,
.code-card:hover .card-header:hover {
    box-shadow: inset 0 0 8px rgba(232, 102, 232, 1), 
                0 0 15px rgba(232, 102, 232, 1);
}

OVERLAY CARD TITLE

Use these properties to create a displaced card title.

           
.card-header {
    position: relative;
    margin-top: -20px;
}
